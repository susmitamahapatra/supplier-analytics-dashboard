# 📊 Supplier Performance & Financial Analytics Dashboard (Power BI)

## 🔍 Project Overview

This project demonstrates an end-to-end Business Intelligence solution built using **Power BI**, focusing on supplier performance and financial analytics.

The dashboard enables stakeholders to monitor spending, identify overdue invoices, and forecast future financial trends using interactive visualizations and automated data processing.

---

## 🎯 Business Problem

Organizations managing multiple suppliers often struggle with:

* Tracking overdue payments and financial exposure
* Monitoring supplier performance
* Generating timely reports
* Lack of predictive insights for future spending

This project solves these challenges by providing a **scalable, automated, and insight-driven dashboard**.

---

## 🛠️ Tools & Technologies

* **Power BI** – Data Visualization & Dashboarding
* **Power Query** – Data Transformation
* **DAX** – Calculations & KPIs
* **CSV Dataset** – Simulated enterprise data

---

## 📂 Dataset

* 1,200+ records simulating real-world supplier transactions
* Includes:

  * Invoice Amounts
  * Payment Status (Paid / Overdue / Pending)
  * Delay Days
  * Ageing Buckets
  * Supplier Categories & Regions

---

## ⚙️ Key Features

### 📊 Executive Dashboard

* KPI Cards:

  * Total Spend
  * Total Invoices
  * Overdue Amount
  * Overdue %
  * Avg Delay Days

* Visuals:

  * Monthly Spend Trend
  * Payment Status Distribution
  * Top Suppliers by Spend
  * Ageing Bucket Analysis

---

### 🔍 Supplier-Level Analysis

* Drill-through functionality
* Supplier-specific performance tracking
* Invoice-level breakdown

---

### 📈 Forecasting (Advanced Feature)

* Implemented **3-month spend forecasting** using Power BI analytics
* Helps anticipate financial trends and future cash flow
